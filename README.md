ZOOX
=====

*Convert a space-delimited text file into .xlsx, .csv, or .tsv.*

Quick install:

```console
~ git clone https://github.com/intertwine/zoox
~ cd zoox/
~ python3 -m venv env
~ . env/bin/activate
~ pip install .
~ zoox --help

Usage: zoox [OPTIONS] FILE_PATH

Options:
  -t, --type [xlsx|csv|tsv]  [default: xlsx]
  --help                     Show this message and exit.
```
